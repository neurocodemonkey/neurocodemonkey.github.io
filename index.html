Projects

BiocSwirl

Open Source

A series of in-depth and easily updateable swirlify made courses used to teach entire bioinformatics workflows in R/Bioconductor using an interactive and easy to digest format. This project was the People's Choice Award for the Vancouver Bioinformatics Hackathon Hackseq2019.

BiocTerm

Open Source

A highly configurable, lightweight and feature rich terminal interface with multi-language support for conducting bioinformatics. BiocTerm currently supports R, Bash, Python, and Vim. This project was the People's Choice Award for the Vancouver Bioinformatics Hackathon Hackseq2019.

behavendoR (early stages)

Open Source

A basic R package used to introduce statistical workflows and reproducibility into the Watson lab and Behavioural Endocrinology field, has basic features and functions that make using R easier for behavioural endocrinologists.
